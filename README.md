# Code Lockdown – Source Code Protection System

## Overview

Code Lockdown is a Python-based mini project designed to protect source code files from unauthorized access. The project uses file handling techniques to lock and unlock source code securely.

## Features

* Lock source code files
* Unlock protected files
* Basic security mechanism
* File handling operations
* Easy command-line execution

## Technologies Used

* Python
* File Handling
* Object-Oriented Programming (OOP)

## Project Structure

* main.py – Main program execution
* locker.py – Source code locking functionality
* decrypt.py – Source code unlocking functionality
* get_serial.py – System identification utility

## How to Run

1. Clone the repository
2. Open the project folder
3. Run:

```bash
python main.py
```
